# 📱 IOSPhoneShop

## 📝 프로젝트 소개
iPhone 제품을 조회하고 구매할 수 있는 iOS 쇼핑몰 애플리케이션입니다. 
REST API를 통해 Spring Boot 백엔드와 연동하여 실시간 데이터를 처리합니다.

## 🛠 Tech Stack
- **Language**: Swift
- **UI Framework**: SwiftUI
- **IDE**: Xcode
- **Backend**: Spring Boot (REST API)
- **Tool**: Wireshark (Traffic Analysis)

## ✨ 주요 기능
- **회원가입 및 로그인**: 사용자 인증 시스템 구축
- **상품 목록 및 상세 조회**: 다양한 iPhone 모델 정보 제공
- **장바구니**: 구매 예정 상품 담기 및 관리
- **마이페이지**: 사용자 정보 관리
